Keyzer 是一款原生应用，用于安全存储你的所有密码及其他私密信息。
在日常生活中，我们都有不希望他人知晓的内容，如信用卡信息、银行账户资料和各类密码。
Keyzer 会将这些数据加密保存为一个可携带的 Keyzer 文件，让你在确保安全的同时，依然能轻松使用。
